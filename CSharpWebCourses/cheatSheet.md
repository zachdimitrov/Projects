## HTTP Protocol
#### Overview
- Hypertext Transfer Protocol
- Communication protocol for the web
- Versions - current HTTP1.1, HTTP2.0
- Request-response actions
- Client- server communication - 7 layers
  - HTTP, TCP, IP, Ethernet, Media...
## HTTP Requests
#### Methods
**Common methods**    
|method|description|
|---|---|
